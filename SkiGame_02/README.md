# Qt 滑雪大冒险

这是一个使用 `Qt 6 Widgets + QPainter + QTimer` 开发的 2D 滑雪小游戏。  
玩家会沿程序化生成的雪坡不断前进，收集金币、躲避岩石，并应对企鹅加速、飞行翅膀、雪人追击、雪崩追击等动态机制。

## 项目特点

- 使用 `QWidget` 自绘，主要画面由 `QPainter` 完成
- 使用 `QTimer` 驱动近似 `60 FPS` 的游戏循环。
- 雪坡不是固定地图，而是由多组正弦函数叠加生成。
- 游戏逻辑已经按面向对象方式拆分为玩家、道具、障碍物、雪人、雪崩等多个类。
- 已接入 `Qt Multimedia`，支持背景音乐、滑雪音效、金币音效、雪人/雪崩追击音效等。

## 游戏状态

游戏状态定义在 `GameWidget::State` 中，共有三种：

- `Menu`
  - 主菜单状态。
  - 显示标题、玩法说明和开始提示。
- `Playing`
  - 正常游戏状态。
  - 执行物理更新、道具生成、碰撞检测和 HUD 绘制。
- `GameOver`
  - 游戏结束状态。
  - 显示本局统计和重新开始提示。

## 核心游戏机制

### 1. 雪坡与移动

- 主角默认会自动向前滑行。
- 雪坡高度通过 `slopeY(x)` 动态生成。
- 坡度通过 `slopeAngle(x)` 近似计算。
- 速度会受到以下因素影响：
  - 前进距离越远，基础速度略微增加。
  - 下坡时速度会变快。
  - 上坡时速度会略微减慢。
  - 跳跃或离地时速度会比贴地滑行略低。
  - 抱住企鹅后会获得额外速度倍率。

### 2. 跳跃

- 支持通过以下输入跳跃：
  - `空格`
  - `上方向键`
  - `W`
  - `鼠标左键`
- 菜单界面点击或按跳跃键，会直接开始游戏。
- 结算界面点击或按跳跃键，会重新开始。

### 3. 金币

- 金币会在前方成组刷新。
- 有时会沿雪坡分布。
- 有时会形成拱形，鼓励玩家跳起来收集。
- 每收集一枚金币：
  - 金币数 `+1`
  - 分数 `+10`
  - 播放 `coin.mp3`
  - 产生金币粒子效果

### 4. 企鹅道具

- 企鹅会在前方随机刷新，频率低于金币。
- 玩家碰到企鹅后：
  - 会抱住企鹅
  - 进入加速状态
  - 速度变快
  - 播放 `penguin.mp3`
- 持续约 `15` 秒后自动结束。
- 持续期间不会再刷新新的企鹅。

### 5. 小翅膀飞行道具

- 翅膀会在前方随机刷新。
- 玩家碰到后进入飞行状态：
  - 脱离雪坡
  - 暂时不会碰到岩石
  - 会播放 `fly.mp3`
- 飞行持续约 `10` 秒。
- 结束后会自然落回雪坡，继续滑雪。
- 飞行和企鹅加速可以同时存在。

### 6. 岩石障碍物

- 岩石会随机刷新在雪坡上。
- 玩家在非飞行状态下撞到岩石会立即死亡。
- 游戏进入 `GameOver`。

### 7. 雪人追击

- 雪人会偶尔从玩家后方出现。
- 同一时间只会存在一个雪人。
- 雪人的追击行为分为不同强度：
  - 有的更偏“施压”
  - 有的更偏“猎杀”
- 雪人会尽量维持在玩家后方的危险距离附近。
- 如果雪人追上玩家，则游戏结束。
- 雪人存在时循环播放 `monster.mp3`。
- 雪人会在追击一段时间后自然消失。

### 8. 雪崩追击

- 雪崩也是从玩家后方出现的追击事件。
- 雪崩与雪人互斥：
  - 同一时刻只会出现一种追击事件
- 雪崩会沿雪坡向前推进。
- 如果雪崩追上玩家，则游戏结束。
- 雪崩存在时循环播放 `avalanche.mp3`。
- 雪崩会在一段时间后自然褪去。

### 9. 出生点场景装饰

- 每局开局时，玩家后方会有一个木屋。
- 木屋附近会插一个小红旗。
- 这些元素属于固定场景装饰，不参与碰撞。

## 画面渲染机制

整个游戏画面由 `GameWidget::paintEvent()` 串联绘制，主要顺序如下：

1. 天空
2. 星星
3. 太阳
4. 云层
5. 远山
6. 树木
7. 雪坡
8. 道具与敌人
9. 障碍物
10. 玩家
11. 粒子效果
12. HUD / 菜单 / 结算界面

## 音频系统说明

音频由 `GameWidget` 中的多个 `QMediaPlayer` 管理：

- 长音频和循环音效：
  - 背景音乐
  - 滑雪音
  - 企鹅音
  - 飞行音
  - 雪人音
  - 雪崩音

- 高频短音效：
  - 金币音效使用“播放器池”轮流播放
  - 这样可以减少连续快速吃金币时的卡顿

## 分数系统

- 距离分数：
  - 玩家前进距离会持续转化为分数
- 金币分数：
  - 每枚金币额外 `+10`
- 总分：
  - `距离分数 + 金币分数`

## 输入说明

- `空格 / 上方向键 / W / 鼠标左键`
  - 跳跃
  - 在菜单或结算界面中也可用于开始/重新开始
- `R`
  - 在结算界面重新开始
- `Esc`
  - 在游玩中直接结束当前游戏

## 构建与运行

### 环境要求

- Qt 6.5 或更高版本
- 支持 `Qt Widgets` 和 `Qt Multimedia`
- CMake 3.19 或更高版本
- 支持 C++ 编译的 Qt 工具链

### 构建步骤

```powershell
cmake -S . -B build
cmake --build build
```

如果你使用的是 Qt Creator，也可以直接打开工程进行构建。
